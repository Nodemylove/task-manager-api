## Tech Stack
Node.js · Express · PostgreSQL · Knex.js · Docker · JWT · Zod · Swagger

## Quick Start
1. Install Docker Desktop
2. Clone this repo
3. cp .env.example .env  (fill in values)
4. docker-compose up -d
5. npm install
6. npm run migrate
7. npm run dev
